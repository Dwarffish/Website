De Opdracht

Wij moesten voor informatica met HTML en CSS een website maken met alles wat we geleerd hadden van Khanacademy en in de les zelf. Aan de website zaten eisen en moest over "digitale beveiliging" gaan.

Eisen

  • De hoofdpagina heeft de naam index.html.
  • Er zijn minimaal vijf zelfgeschreven html pagina’s per persoon.
  • Alle pagina’s moeten via het menu aan elkaar gekoppeld zijn.
  • Het menu is gemaakt met een ordered of unordered list en list items.
  • Alle tags die je in de les geleerd hebt, moet je gebruiken in je website (o.a. header, footer, section, nav, ....).
  • Alle opmaak en layout moet via CSS gerealiseerd worden.
  • Je gaat gebruik maken van een extern CSS bestand.
  • Alle HTML-code voldoet aan de voorschriften die gesteld worden aan HTML5.
  • De website is liquid, dat wil zeggen dat hij altijd schermvullend is op elke resolutie.
  • Er wordt geen gebruik gemaakt van frames maar van het div element om de pagina op te delen.
  • Er is één pagina met een formulier dat ingevuld kan worden.
  • Er mogen maximaal twee youtube-video's in alle pagina´s samen ge-embed worden. Deze video's moeten aanvullend zijn op de informatie die al op diezelfde pagina staat.
  • Plaats op slechts één pagina (de kortste) een mogelijkheid die pagina voor te laten lezen via erbij horend mp3-bestand.
